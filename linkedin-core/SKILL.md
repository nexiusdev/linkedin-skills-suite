# LinkedIn Core

This skill stores the shared runtime files used by the LinkedIn skill suite:

- `linkedin-core/shared/logs/`
- `linkedin-core/shared/scripts/`
- `linkedin-core/shared/references/`
- `linkedin-core/references/`

All other skills should reference this folder for cross-skill state and global references.
