## Prospect Pipeline

### Profile Cache

**Purpose:** Cache LinkedIn profile data to minimize profile visits and respect daily limits.

**Update Rule:** Update cache every time you visit a prospect's LinkedIn profile.

| Name | Profile URL | Last Checked | Activity Status | Last Post Date | Recent Post URLs |
|------|-------------|--------------|-----------------|----------------|------------------|

**Activity Status Values:**
- ACTIVE: Posted in last 7 days
- MODERATE: Posted in last 30 days
- INACTIVE: No posts in 30+ days
